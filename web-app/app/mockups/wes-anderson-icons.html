Design Principles

  • All icons use geometric shapes for clarity and vintage aesthetic
  • Maximum 2-3 colors per icon to maintain visual cohesion
  • Symmetry is prioritized when conceptually appropriate
  • 2-3px stroke weights maintain legibility at all sizes
  • Inline SVG for easy color customization and performance
  • Icons scale cleanly from 40px to 120px without losing detail

Usage Guidelines

  • Primary icons at 60-80px for feature cards
  • Utility icons at 40-50px for inline usage
  • Media type icons at 30-40px for badges and labels
  • Maintain consistent spacing: 15-20px between icon and text
  • Use bottle green and mid-blue for accent elements
  • Never use more than 3 accent colors in a single component

Accessibility Notes

  • All icons have 3:1 contrast minimum against backgrounds
  • Icons always paired with text labels for clarity
  • Color never used as the sole means of conveying information
  • Focus states include visible outline for keyboard navigation
  • ARIA labels provided for icon-only interactive elements
