# Fictotum Icon Design Brief for Gemini

## Project Overview

You are creating a custom icon library for **Fictotum**, a historical data visualization platform. The icons must embody a **Wes Anderson aesthetic**: geometric, symmetrical, vintage (1960s-70s), whimsical yet precise.

---

## Design Requirements

### Visual Style
- **Aesthetic**: Wes Anderson films (The Grand Budapest Hotel, Moonrise Kingdom, The French Dispatch)
- **Form**: Geometric shapes only (circles, rectangles, triangles, lines)
- **Stroke Weight**: 2-3px consistent throughout each icon
- **Symmetry**: Symmetrical when conceptually appropriate
- **Complexity**: Simple enough to read at 24px, detailed enough to be charming at 64px
- **Era**: Vintage 1960s-70s travel poster / scout badge / passport stamp aesthetic

### Technical Specifications
- **Format**: SVG (inline, no external dependencies)
- **Size**: Design at 64px √ó 64px artboard
- **Colors**: Use 2-3 colors maximum per icon from the palette below
- **Line Style**: Clean, geometric, no gradients
- **Background**: Transparent

## Notes on Literalism

The user has emphasized that icons must be **immediately recognizable**. Avoid:
- Overly abstract symbols
- Ambiguous shapes that could represent multiple things
- Clever conceptual metaphors that require explanation

Favor:
- Literal representations of the object/concept
- Clear silhouettes and shapes
- Distinctive features that define the icon's subject
- Details that aid recognition (sprocket holes on film, spokes on reels, etc.)

**Test**: Show the icon to someone unfamiliar with the context. Can they name what it represents in under 2 seconds?
